HTML 結構說明

<!DOCTYPE HTML>
Document type的縮寫。文檔類型聲明，表示該文件為HTML5文件。<!DOCTYPE>聲明必須是HTML文檔的第一行，位於<html>標籤之前。

<html lang="zh-Hant-TW"> 
lang是標註網頁的語系,讓瀏覽器能更正確的解析與編碼。
<head>
<meta charset="UTF-8">
charset=UTF-8是告知瀏覽器此頁面屬於什麼字元編碼格式，下一步瀏覽器做好「翻譯」工作。

<meta name="author" content="name">
標註本網頁作者姓名等資料。

<meta name="robots" content="all">
控制搜尋引擎檢索及建立索引的行為。

<meta name="description" content="網頁說明">
提供簡短的網頁說明。

<meta name="keywords" content="網頁說明">
讓搜索引擎容易找到；這個屬性在以前是能幫助到網站排名，但現在google早已經拋棄這個屬性，並不把他列出排名的參考，用description即可。

<link rel="stylesheet" href="style.css" type="text/css">
relationship關聯;Hypertext超文字;Reference出處。

<link rel="stylesheet" href="fontawesome/css/all.css" type="text/css">
在網頁加入Font Awesome的ICON圖示。

<link rel="icon" href="favicon.ico">
網站logo。favicon是Favorites Icon的縮寫。

<title>網頁標題</title>
</head>
<body>
</body>
</html>
